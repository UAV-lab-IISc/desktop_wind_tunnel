# 3D Printed Components

## Load Cell Mounting and Support Structure

---

## Overview

This folder contains the 3D printable components used to mount and support the load cells and airfoil model inside the wind tunnel test section.

The mechanical structure ensures:

* Stable force measurement
* Proper alignment of load cells
* Minimal vibration
* Reliable aerodynamic testing

These components are essential for accurate lift and drag measurement.

---

## Components Included

Load Cell Mount

Load Cell Stand

Load Cell Holder

Base Plate

These parts form the mechanical support system for the L-shaped load cell configuration used in the wind tunnel.

---

## File Format

All models are provided in:

STL (Standard Tessellation Language)

Compatible with:

* Cura
* PrusaSlicer
* Bambu Studio
* Any FDM 3D printer

---

## Recommended Print Settings

Material:

PLA or PETG

Layer Height:

0.2 mm

Infill:

30 to 40 percent

Supports:

Enabled

Print Orientation:

Flat base on build plate

---

## Assembly Notes

Ensure the load cells are mounted rigidly.

Avoid loose connections.

Align the airfoil vertically.

Secure all screws tightly.

Verify stability before calibration.

---

## Purpose

These components provide the mechanical foundation required for:

* Lift and drag measurement
* Stable experimental testing
* Reliable sensor readings
* Safe wind tunnel operation
